## Context

当前模式选择器使用 Card 包裹 ListWheelScrollView，占据一整行。用户希望将其改为磁贴形式，更紧凑且与状态磁贴风格统一。

## Goals / Non-Goals

**Goals:**
- 模式选择器改为 2x2 磁贴（占 4 个标准磁贴位置）
- 保留滚轮选择交互
- 统一 UI 风格

**Non-Goals:**
- 不改变选择器的交互逻辑
- 不改变命令发送时机

## Decisions

1. **磁贴大小**: 2x2（横向占 2 格，纵向占 2 格）
   - 理由：滚轮选择器需要足够高度来显示多个选项

2. **布局方式**: 使用 StaggeredGridView 或自定义布局
   - 理由：标准 GridView 不支持不同大小的子项

3. **磁贴位置**: 放在录制控制磁贴后面
   - 理由：保持功能分组合理

## Risks / Trade-offs

- 2x2 磁贴可能导致网格布局复杂 → 使用 Wrap 或自定义布局简化
