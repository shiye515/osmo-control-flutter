## Context

当前工作台页面有一个独立的录制控制卡片，包含"开始录制/停止录制"按钮和"拍照"按钮。用户希望将此功能整合到状态磁贴网格中。

## Goals / Non-Goals

**Goals:**
- 将录制/拍照功能整合为一个磁贴
- 根据相机状态自动切换功能
- 减少页面空间占用

**Non-Goals:**
- 不改变协议逻辑
- 不添加新的控制功能

## Decisions

1. **磁贴位置**: 作为状态磁贴网格的第一个磁贴
   - 理由：最重要的控制功能应该最显眼

2. **状态判断逻辑**:
   - `isRecording == true` → 停止录制（红色图标）
   - `cameraMode == 0x05 (拍照)` → 拍照（蓝色图标）
   - 其他情况 → 开始录制（蓝色图标）

## Risks / Trade-offs

- 用户可能不清楚磁贴的功能 → 通过图标和文字清晰表达当前状态
