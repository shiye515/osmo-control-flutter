## Context

osmo-control-flutter 是一个用 Flutter 开发的 DJI Osmo 相机控制 App，已完成核心功能（蓝牙连接、相机控制、GPS 推送），并上架 Android 版，iOS 版等审核。当前缺乏一篇面向目标用户（DJI Osmo 用户、Flutter 开发者）的推广文章。

## Goals / Non-Goals

**Goals:**
- 撰写一篇 Markdown 推广文章，放在项目根目录 `publish.md`
- 文章结构：50% SDD 开发过程叙事 + 50% App 功能详细介绍
- 语言生动有感染力，面向中文技术用户

**Non-Goals:**
- 不修改任何代码
- 不修改 README.md

## Decisions

**决策 1：文章放置位置**
- 放在项目根目录 `publish.md`，与 README.md 并列，便于直接引用
- 备选：放在 `docs/` 目录 → 不选，因为根目录更显眼

**决策 2：文章结构**
- 上半部分：SDD 开发过程（时间线叙事：2026-03-29 到 2026-04-01，共 4 天）
- 下半部分：功能详细介绍（设备连接、状态监控、拍摄控制、GPS 推送）
- 开头：吸引眼球的 Hook、痛点描述

**决策 3：SDD 介绍方式**
- 通过具体 spec 文件和提交记录来佐证 SDD 的效果
- 展示 spec → 实现 的工作流，配合实际截图文字描述

## Risks / Trade-offs

- [风险] 文章过于技术化，普通用户难以理解 → 缓解：分层写作，功能部分用通俗语言
- [风险] 文章过于营销化，开发者反感 → 缓解：以真实数据（提交记录、时间线）增加可信度
